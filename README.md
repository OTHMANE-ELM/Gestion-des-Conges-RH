# 🧑‍💼 RH Congés Management System

## 📖 Description

Application full-stack de gestion des congés des employés développée avec **Spring Boot** (backend) et **Thymeleaf** 

Ce projet permet de :
- Gérer les employés
- Gérer les demandes de congés
- Appliquer des règles de validation (admin / directeur / secrétaire)
- Sécuriser l'application avec JWT / Spring Security

---

## 🚀 Technologies utilisées

### 🔙 Backend
- Spring Boot
- Spring Security
- Thymeleaf
- Spring Data JPA / Hibernate
- MySQL
- Maven



## 📌 L'application démarre sur :      http://localhost:8081
